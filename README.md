# Resipal

Resipal is a complete community management platform built with Flutter and Supabase. It provides tools to manage residential properties, financial ledgers, and member directories across three dedicated applications.

## Applications

### Admin
* **Property Management:** Unit registry and fee tracking.
* **Financial Control:** Payment verification and ledger oversight.
* **User Administration:** Management of invitations and community roles.

### Resident
* **Payments:** Digital submission of maintenance fees and history tracking.
* **Invitations:** Generation of access codes for guests and service providers.
* **Communication:** Direct access to community announcements and documents.

### Security
* **Access Control:** Validation of resident and visitor credentials.
* **Registry:** Real-time logging of entries and exits.
* **Alerts:** Immediate notification of security incidents.

## Tech Stack
* Flutter
* Supabase (Database, Edge Functions)
* WhatsApp Business API (for automated invitations)
